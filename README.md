# 🏥 Projeto Clínica EcoVida

Este projeto faz parte de um desafio prático da **Digital Innovation One (DIO)**, focado na estruturação de páginas web utilizando apenas **HTML5** (sem a aplicação de CSS externo, conforme os requisitos do módulo).

O objetivo é simular o site de uma clínica médica que integra tecnologia e sustentabilidade, aplicando conceitos fundamentais de desenvolvimento web.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica de todo o conteúdo.
* **Tabelas**: Utilizadas para o layout (template) e para a grade de horários de atendimento.
* **Formulários**: Implementação de coleta de dados na página de contato.
* **Mídias**: Incorporação de imagens e Mapas (Iframe).

## 📂 Estrutura do Projeto

O site é composto por 4 páginas principais:

1.  **Página Principal (`index.html`)**: Introdução e missão da clínica.
2.  **Sobre a Clínica (`sobre.html`)**: História e diferenciais sustentáveis.
3.  **Horário de Atendimento (`horario.html`)**: Tabela detalhando os serviços e horários por dia da semana.
4.  **Contato (`contato.html`)**: Informações de localização, mapa interativo e formulário de agendamento/contato.
